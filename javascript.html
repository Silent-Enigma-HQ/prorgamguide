Что такое JavaScript?

JavaScript — это язык программирования для добавления интерактивности на веб-страницы. Он используется для создания динамичных сайтов, а также для разработки веб-приложений и серверных решений.

Пример программы


let name = "Мир";
console.log("Привет, " + name + "!");

Основные понятия и синтаксис

Переменные и типы данных

Переменные в JavaScript могут быть объявлены с помощью var, let или const. Типы данных включают числа, строки, булевы значения, массивы и объекты.

Операторы

JavaScript поддерживает арифметические, логические и условные операторы. Например, для сравнения можно использовать операторы == и ===.

Функции

Функции — это блоки кода, которые выполняют определенную задачу. Они могут принимать параметры и возвращать значения. Например:


function greet(name) {
    return "Привет, " + name + "!";
}
console.log(greet("Мир"));

Массивы и объекты

Массивы используются для хранения упорядоченных данных, а объекты — для хранения данных в виде пар "ключ-значение".


let array = [1, 2, 3];
let obj = { name: "JavaScript", type: "программирование" };

Работа с DOM

Document Object Model (DOM) позволяет взаимодействовать с HTML-страницей через JavaScript. С помощью DOM можно изменять элементы страницы, добавлять события и анимации.

Асинхронность в JavaScript

Для работы с асинхронными операциями, такими как загрузка данных, используются callback-функции, промисы и синтаксис async/await.


async function fetchData() {
    let response = await fetch('https://api.example.com');
    let data = await response.json();
    console.log(data);
}

Что такое Python?

Python — это высокоуровневый язык программирования, известный своей простотой и читаемостью.

Он широко используется в разных областях, от веб-разработки до анализа данных и машинного обучения.

Как установить Python

Для начала работы с Python, его необходимо установить на вашем компьютере.

  • Зайдите на официальный сайт Python и выберите версию для вашей операционной системы (Windows, macOS, Linux).
  • Установите Python, следуя инструкциям на экране.
  • Для удобства работы можно установить IDE, например, PyCharm или VSCode.

Первая программа на Python

После установки Python, можно написать свою первую программу, которая выведет строку на экран.


# Это комментарий
print("Привет, мир!")

Основные типы данных в Python

Python поддерживает несколько базовых типов данных:

  • Числа: целые числа и числа с плавающей точкой (например, 5, 3.14).
  • Строки: последовательности символов, заключенные в кавычки (например, "Python").
  • Списки: упорядоченные коллекции элементов (например, [1, 2, 3]).
  • Словари: коллекции пар "ключ-значение" (например, {"name": "Alice", "age": 25}).

Переменные и операторы

Переменные в Python используются для хранения значений, которые можно изменять и использовать в различных операциях.

Пример:


a = 5
b = 10
print(a + b)  # Результат: 15

Условия и циклы

В Python можно использовать условные операторы для принятия решений:


if a > b:
    print("a больше b")
else:
    print("a не больше b")

Также доступны циклы, например, цикл for для перебора элементов в коллекции:


for i in range(5):
    print(i)  # Выведет числа от 0 до 4

Функции

Функции позволяют организовать код и повторно использовать его.


def greet(name):
    print("Привет, " + name + "!")

greet("Иван")  # Выведет: Привет, Иван!

Ошибки и исключения

Иногда в процессе работы с программой могут возникать ошибки. Python позволяет их обрабатывать с помощью блока try-except.


try:
    x = 10 / 0
except ZeroDivisionError:
    print("Невозможно разделить на ноль")

Работа с файлами

Python позволяет работать с файлами, например, читать и записывать данные в текстовые файлы.


with open("example.txt", "w") as file:
    file.write("Привет, Python!")

Основы ООП в Python

Python поддерживает объектно-ориентированное программирование. Вот пример создания класса:


class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def speak(self):
        return f"{self.name} говорит гав!"

dog = Dog("Бобик", 3)
print(dog.speak())  # Выведет: Бобик говорит гав!

Примеры проектов на HTML

Проект 1: Простая веб-страница

<!DOCTYPE html>
<html>
<head>
    <title>Простая веб-страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Это простая веб-страница.</p>
</body>
</html>

Пример создания базовой HTML-страницы.

Проект 2: Форма ввода данных


<form>
    <label for="name">Ваше имя:</label>
    <input type="text" id="name" name="name"><br>
    <label for="email">Ваш email:</label>
    <input type="email" id="email" name="email"><br>
    <input type="submit" value="Отправить">
</form>

Пример создания простой формы ввода данных.

Проект 3: Вставка изображения


<img src="image.jpg" alt="Описание изображения">

Пример вставки изображения на веб-страницу с атрибутами src и alt.

Проект 4: Встраивание видео


<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Пример встраивания видео с YouTube на страницу.

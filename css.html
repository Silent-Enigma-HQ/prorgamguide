Что такое CSS?

CSS (Cascading Style Sheets) используется для оформления веб-страниц: цвета, шрифты, расположение блоков и т.д. CSS позволяет отделить структуру веб-страницы от её оформления, создавая более гибкий и удобный код.

Основные понятия CSS

  • Селекторы — определяют, к каким элементам применяется стиль.
  • Свойства и значения — каждое правило CSS состоит из свойства и его значения. Например, color: red;.
  • Каскадирование и наследование — правила каскадируются, то есть, если два правила применяются к одному элементу, то применяется последнее по порядку, если не указано иначе.
  • Приоритеты правил — правила с более специфичными селекторами имеют больший приоритет.

Основные свойства CSS

  • Шрифты: можно изменить тип шрифта, размер и толщину. Пример: font-family: Arial, sans-serif;
  • Цвета и фоны: для изменения цвета текста и фона. Пример: background-color: #f0f0f0;
  • Модели блочных элементов: каждая область на странице (блок) имеет отступы (margin), рамки (border), внутренние отступы (padding).
  • Позиционирование: позволяет задавать точное расположение элементов. Пример: position: absolute;

Сеточные системы и выравнивание

С помощью Flexbox и CSS Grid можно создавать сложные макеты и выравнивать элементы на странице.

  • Flexbox: используется для выравнивания элементов по горизонтали или вертикали.
  • CSS Grid: позволяет создавать сетку для более сложных макетов с несколькими колонками и строками.

Медиа-запросы и адаптивный дизайн

Медиа-запросы позволяют создавать страницы, которые адаптируются под различные устройства (например, мобильные телефоны, планшеты, компьютеры).


@media screen and (max-width: 768px) {
    body {
        background-color: lightblue;
    }
}

Трансформации и анимации

  • CSS-трансформации: позволяют изменять элементы на странице (поворот, масштабирование, перемещение).
  • CSS-анимирование: можно задавать анимации для различных свойств, таких как прозрачность, положение и другие.

Пример кода

Вот пример базового CSS-кода для стилизации страницы:


body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #4a90e2;
    text-align: center;
}

Пример кода с Flexbox

Вот пример использования Flexbox для выравнивания элементов по горизонтали и вертикали:


.container {
    display: flex;
    justify-content: center; /* Выравнивание по горизонтали */
    align-items: center; /* Выравнивание по вертикали */
    height: 100vh;
}

.box {
    width: 100px;
    height: 100px;
    background-color: #4a90e2;
    color: white;
    display: flex;
    justify-content: center;
    align-items: center;
}

Практические примеры

  • Стилизация кнопки: можно добавить стили для кнопок, чтобы они выглядели более привлекательными.
  • Стилизация карточек: с помощью CSS можно создать красивые карточки с изображениями и текстом.
  • Навигационное меню: пример горизонтального меню с использованием CSS.

Часто встречающиеся ошибки

  • Забытые точки с запятой: всегда завершаем строки свойств точкой с запятой.
  • Неправильные значения свойств: например, неправильный цвет или размер шрифта.
  • Ошибки при использовании селекторов: неправильное использование классов и id.
  • Проблемы с адаптивностью: элементы могут не подстраиваться под различные размеры экранов.

Примеры проектов на CSS

Проект 1: Базовые стили

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
}

Пример применения основных стилей для фона и шрифта.

Проект 2: Центрирование текста

.center {
    text-align: center;
}

Пример использования свойства text-align: center для центрирования текста.

Проект 3: Flexbox для выравнивания

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

.box {
    width: 100px;
    height: 100px;
    background-color: #4CAF50;
}

Использование Flexbox для центрирования элементов на странице.

Проект 4: CSS Grid

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

.item {
    background-color: #4CAF50;
    padding: 20px;
}

Пример использования CSS Grid для создания сетки элементов.

Проект 5: Анимация при прокрутке

.element {
    opacity: 0;
    transform: translateY(20px);
    transition: opacity 0.3s, transform 0.3s;
}

.element.visible {
    opacity: 1;
    transform: translateY(0);
}

Анимация элементов при прокрутке страницы.

Проект 6: Интерактивное меню

nav ul {
    list-style-type: none;
    padding: 0;
}

nav ul li {
    display: inline-block;
    margin: 10px;
}

nav ul li a {
    text-decoration: none;
    color: #333;
    padding: 10px;
    transition: background-color 0.3s;
}

nav ul li a:hover {
    background-color: #4CAF50;
}

Меню с анимацией при наведении.

Проект 7: Модальные окна

.modal {
    display: none;
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.5);
}

.modal-content {
    position: absolute;
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);
    background-color: #fff;
    padding: 20px;
}

Модальные окна, которые можно открывать и закрывать с помощью CSS.

Проект 8: Карточки с эффектами на наведении

.card {
    width: 200px;
    height: 300px;
    background-color: #f4f4f4;
    transition: transform 0.3s, box-shadow 0.3s;
}

.card:hover {
    transform: translateY(-10px);
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
}

Карточки с анимацией при наведении курсора.

Проект 9: Параллакс эффект

.section {
    background-image: url('background.jpg');
    background-attachment: fixed;
    background-position: center;
    background-size: cover;
}

Параллакс эффект для фона с помощью CSS.

Проект 10: Анимации с @keyframes

@keyframes example {
    0% {
        transform: rotate(0deg);
    }
    100% {
        transform: rotate(360deg);
    }
}

.element {
    animation: example 2s infinite;
}

Анимации, созданные с помощью @keyframes.

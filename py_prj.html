Проект 4: Игра "Угадай число"


import random

secret_number = random.randint(1, 10)
guess = int(input("Угадайте число от 1 до 10: "))

if guess == secret_number:
    print("Вы угадали!")
else:
    print("Попробуйте снова!")

Простая игра, в которой нужно угадать случайное число от 1 до 10.

Проект 6: Простой веб-сервер на Flask

Проект 7: Парсер данных с использованием BeautifulSoup


import requests
from bs4 import BeautifulSoup

url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, "html.parser")

# Найти все ссылки на странице
links = soup.find_all("a")
for link in links:
    print(link.get("href"))

Пример парсинга данных с веб-страницы и извлечения всех ссылок.

Проект 8: Калькулятор с графическим интерфейсом на Tkinter


import tkinter as tk

def add():
    result.set(float(entry1.get()) + float(entry2.get()))

window = tk.Tk()

entry1 = tk.Entry(window)
entry1.pack()

entry2 = tk.Entry(window)
entry2.pack()

result = tk.StringVar()
label = tk.Label(window, textvariable=result)
label.pack()

button = tk.Button(window, text="Сложить", command=add)
button.pack()

window.mainloop()

Пример простого калькулятора с графическим интерфейсом, использующий Tkinter.

Проект 9: Программа для управления задачами (ToDo List)


tasks = []

def add_task(task):
    tasks.append(task)

def show_tasks():
    for task in tasks:
        print(task)

add_task("Купить продукты")
add_task("Написать отчет")

show_tasks()

Пример программы для отслеживания задач с добавлением и выводом списка задач.

Проект 10: Простой чат-бот на Telegram


import telegram

bot = telegram.Bot(token="YOUR_API_KEY")

updates = bot.get_updates()
for update in updates:
    print(update.message.text)

Простой чат-бот для Telegram, который выводит текст сообщений пользователей.
